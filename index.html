How to play

Aim of the game:

Unscramble the 5 words that are hidden on the board. Each row should contain a valid 5 letter word. You will have to swap letters across the rows and columns to find all five words.

Board:

The game board is made up of 25 squares, formed into a 5 x 5 grid.

How:

Swap letters by clicking/pressing on 2 letters you want to swap (one after another)

How do i know if it's correct?:

Click the check button when you are happy with how the board is looking. It should end up looking something like below.

What next?:

The squares will light up with 3 different colours to help guide you to the solution

  • Green: This means you have the correct letter in the correct spot, do not move this anymore

  • Orange: This means you have a letter that is in the correct row, but not the correct column

  • Pink: This means you have a letter in the correct column, but not the correct row

Tips:

  • The order of checking is for correctly positioned letters (Green), then correct row but incorrect column (Orange) and finally the correct column but incorrect row (Pink). So take care!
  • Try to make as many letter swaps as you feel comfortable doing inbetween each check, this will minimise checks and make you look like a Pro
  • Each word has to be in the correct row, but the check validation will help with that
